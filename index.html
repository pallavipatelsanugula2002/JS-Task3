Datatypes in JS

number

javascript has only one type of number,number can be written with or without decimals. if you add a number and string,we will get a string. numbers are immutable which means we can't change literal values

String

String is 0 or more characters written within the strings. Strings are used to store and manipulating texts. strings are immutable which means we can't change literal values

Boolean

Boolean datatype can take the values as either true or false.

undefined

If a number has been declared but we didn't assign any value to it. it will take as undefined.

array

array data type is used when we want to store multiple data items into a single variable.not to waste the space and memory. arrays can also take the values as null and undefined. Arrays are mutable,which means we can change the values in an array.

Objects

In arrays when we want to access the data items,we can do it through indexing. but when we have multiple data items and if we want to retrieve a single dataitem from it.it is hard to remember all the specific index values,to overcome that problem JS has inroduced the concept of objects. Instead of giving direct values we will give them as key-value pairs.so that we can access the values by key instead of indexing concept
